# StockSalad

## ✏️ 프로젝트 소개
관심있는 주식 상품들에 대한 정보를 한 눈에 볼 수 있는 웹사이트입니다. 일별 가격 변동 그래프, 뉴스 등을 제공합니다. 또한, 사용자가 원하는 주식 상품들을 포트폴리오에 추가하여 관리할 수 있습니다.
